# slackapp-chatbot

このリポジトリは、Python と OpenAI API を使用して作成した **Slack 用 AI チャットボット** のサンプルプロジェクトです。

This repository is a sample project of an **AI chatbot for Slack** created using Python and OpenAI API.

---

## 概要 / Overview

### 日本語
- Python を使用し、Slack の Bot ユーザーと連携する AI チャットボットを開発しました。
- OpenAI の API (ChatGPT) を利用して、Slack 上で対話できる機能を実装しています。

### English
- We developed an AI chatbot that works with Slack's Bot user using Python.
- We implemented features that enable interactive conversations on Slack by utilizing OpenAI's API (ChatGPT).

---

## 使用技術 / Technologies Used

- Python
- OpenAI API (ChatGPT)
